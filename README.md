# superstore-bi-analytics

End to end BI and analytics project. Using Superstore sales data for this project.

## Business Questions
1. How have sales, profit, and order volume evolved over time?
2. Which products and categories drive the majority of revenue and profit?
3. Which customers generate the highest value?
4. How does discounting impact profitability across products and segements?
5. Are there customer segements with high sales but low or negative profit?
6. Which regions and markets underperform and why?
7. How concentrated is the revenue?
8. What actions can be made to improve the profit?

## Project scope
- Data cleaning and transformation using Python
- Dimensional data modeling
- SQL analytics and KPIs
- Customer and profit analysis
- Business focused insights and recommendations
